Build your Properties views, no code needed

EEF is a presentation framework for the Eclipse Modeling Framework (EMF). It allows users to create rich user interfaces to edit EMF models.

Declarative

No need to generate code, everything is interpreted from a description of your user interface

Simple

Your first user interface from scratch in a couple of minutes

Configurable

Customize everything easily with very powerful interpreters

EEF + Sirius

Eclipse Sirius provides a kickass bridge to create very easily EEF-based Properties views for your designers. Find more out the integration of the new EEF runtime in Eclipse Sirius in the Sirius documentation
