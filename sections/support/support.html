Community Support

Just like any other project of the Eclipse Foundation, you can find support from the Eclipse EEF community in the various EEF community channels

Forum

Looking for an answer? Find all the questions and answers of the Eclipse EEF community on our dedicated forum

Found a bug? Don't forget to search the Eclipse EEF Bugzilla to find out if it has already been reported or report it to help the project
